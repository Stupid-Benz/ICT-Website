To print "Hello World", it is much more complicated than Python in .NET 5 or before.


using System;

namespace HelloWorld
{
  class Program
  {
    static void Main(string[] args)
    {
      Console.WriteLine("Hello World!");    
    }
  }
}

But starting from .NET 6, C# console app template generates top-level statements, Python like statements. 


Console.WriteLine("Hello, World!");
